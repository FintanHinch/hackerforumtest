# Hacker News Front Page

Small React + Typescript webpage that displays Hacker News stories using the offical Firebase API

# Features :

            - Toggle Between **Top** and **New Stories
            - Pagination using a load more button
            - Test coverage

# Tech

            - React Typescript Vite
            - TanStack React Query
            - SCSS
            - Vitest + Testing Library


#Future Improvements
            - Add barrel files for cleaner imports

            
# Setup

npm i
npm run dev
